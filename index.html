Weather Analysis with OpenWeatherMap API and Python

Project Overview

This was a useful project showcasing Python's capabilities to access data via API calls, cleansing and analysis with Pandas, and visualization with Pandas native visualization capabilities and Matplotlib.

The weather data was collecting using the OpenWeatherMap API to collect weather data on 500 randomly chosen cities around the world on May 15, 2018 at 1:30am EDT. The Pandas Python library was used to connect to the OpenWeatherMap API and cleanse the data. Pandas native visualization and Python's Matplotlib visualization library were used to create scatter plots graphing the parameters by latitude for each city.

This random sample (and others ran contemporaneously) have a greater density of cities in latitudes above the equator than below. One should take this into account for future analyses. For example, it appears from these graphs that humidity and cloud cover increase as latitude increases, but this doesn't seem to make logical sense on its face. Further, this analysis did not control for many factors - including current season (depending on longitude) and time of data collected.
